# MindSpace: Digital Wellness & Mental Health Platform

## Overview

MindSpace is a PHP web application designed to promote digital wellness and mental health.
The platform gives users helpful wellness resources, a mood journal, guided breathing exercises, daily affirmations, and a wellness summary.
This project was developed using HTML, CSS, and PHP.

## Features

### Mental Wellness Resources
- Resource directory generated from a PHP array
- Displayed dynamically using a foreach loop
- Links to mental health and wellness resources

### Mood Journal
- Journal entry form using $_POST
- Stores the user's current mood using $_SESSION
- Displays previous journal entries
- Mood slider with emoji feedback

### Guided Breathing Exercise
- CSS @keyframes breathing animation
- Step-by-step breathing instructions
- Responsive layout

### Daily Affirmation Carousel
- CSS-only carousel
- Uses :checked radio buttons
- No JavaScript required

### Wellness Summary
- Displays the user's latest mood
- Daily wellness tip selected from a PHP array
- Wellness streak section
- Relaxing cat slideshow
